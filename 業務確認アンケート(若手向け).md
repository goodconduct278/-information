# 業務知識・引継ぎ状況アンケート 完成版

作成日：2026年5月20日  
用途：業務知識DB作成・ベテラン社員ヒアリング前の情報収集  
対象：入社1〜5年目までの社員  
目的：若手社員が実務上困っている業務、資料だけでは判断しづらい業務、ベテラン社員に確認したい内容を整理する

---

## 1. 本アンケートの目的

本アンケートは、入社1〜5年目までの社員が日々の業務で感じている、以下の内容を整理するためのものです。

- 自分だけで進めるには不安がある業務
- 資料やマニュアルだけでは判断しづらい業務
- ベテラン社員に確認しないと判断しづらい内容
- 後輩や後任者に説明しづらい内容
- 過去の経緯を知らないと対応しづらい業務
- 客先・取引先ごとの注意点
- 業務知識DBに残すべき情報

回答内容は、今後の業務知識DB、手順書、教育資料、引継ぎ資料、ベテラン社員へのヒアリング項目の作成に活用します。

本アンケートは、特定の個人や業務を問題視するものではありません。  
業務を安定して継続し、後任者や若手が困らない状態を作ることを目的としています。

# 13. 回答後の整理方法

アンケート回収後は、回答を以下の分類で整理します。

| 分類 | 内容 |
|---|---|
| 判断基準系 | どの条件ならどう判断するか |
| 過去経緯系 | なぜ今の運用になったか |
| 客先注意点系 | 客先ごとの癖・過去トラブル・対応方法 |
| トラブル防止系 | 過去のミス・手戻り・再発防止 |
| 資料探索系 | どの資料を見ればよいか、どこにあるか |
| 確認先系 | 誰に聞けばよいか、どの部署を通すか |
| 教育系 | 後輩に説明しづらいこと |
| DB化優先候補 | 早めに業務知識DB化すべき業務 |

---

# 14. ベテランヒアリングへの変換例

このアンケートの回答をもとに、ベテラン社員への聞き取り項目へ変換します。

| 若手社員の回答 | ベテランへの質問 |
|---|---|
| 特殊案件の見積判断が不安 | 特殊案件の見積では、何を基準に判断していますか |
| 客先ごとの対応方法が分からない | 客先ごとに注意していることはありますか |
| 過去案件の経緯を知らないと判断できない | 後任者が知っておくべき過去経緯はありますか |
| どの資料が正しいか分からない | 参考にすべき資料・使ってはいけない古い資料はありますか |
| 後輩に説明しづらい | 新人・後任者に必ず教えるべきポイントは何ですか |

# 15. 実施後の流れ

```text
1. 入社1〜5年目社員へ本アンケートを実施する
2. 回答を分類する
3. 業務知識DB化・ヒアリング対象業務を3〜5件選ぶ
4. ベテラン社員へ確認すべき質問を整理する
5. ベテラン社員へヒアリングを行う
6. 聞き取り内容を業務知識DBの試作ページにまとめる
7. 実際に若手・後任者が使えるか確認する
8. 不足している情報を追加する
```

# 16. 上司説明用の一言

本アンケートは、入社1〜5年目社員が実務上困っている内容や、ベテラン社員に確認したい内容を洗い出すためのものです。  
簡易版・詳細版に分けず、まず全体の業務を整理したうえで、特に判断が難しい業務・資料だけでは対応しづらい業務・後輩に説明しづらい業務を抽出します。  
その回答をもとに、ベテラン社員へのヒアリング項目を作成し、業務知識DB化する優先業務を選定します。

---

# 17. 一言でのまとめ

このアンケートの目的は、若手社員の「分からないこと」を集めることだけではありません。

目的は、実務上必要だが資料に残っていない知識を見つけ、ベテラン社員への聞き取りを通じて、会社に残すべき知識へ変換することです。
